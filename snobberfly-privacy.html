Snobberfly Privacy Policy

Effective date: July 21, 2026

Contact: erin@beadsnob.com

Snobberfly is an internal business-analytics tool built and operated by BeadSnob (Etsy shop

BeadSnobBoutique). It is used solely by the shop's own two operators. It is not a public

product, has no third-party users, and offers no sign-up.

What the app does

Snobberfly connects to the shop's own accounts (Etsy, Pinterest, Instagram) via each

platform's official API, using read-only permissions, to correlate the shop's own posting

activity with its own sales.

Data we access

  • Pinterest: the shop's own pins, boards, and pin analytics (impressions, saves, clicks)

via the Pinterest API v5 with read-only scopes (pins:read, boards:read,

user_accounts:read).

  • The app never reads, collects, or stores data about any Pinterest user other than the shop's

own business account. It does not create, modify, or delete any content.

How data is stored and used

  • Data is stored in a private local database on hardware owned by the shop, accessible only to

the shop's operators over an encrypted private network. It is not hosted publicly.

  • Data is used exclusively for the shop's internal analytics (e.g., "did sales rise after this

pin?"). It is never sold, shared, transferred, or disclosed to any third party.

  • OAuth tokens are stored locally and used only to make API requests on the shop's behalf.

Data deletion

Disconnecting the app (or revoking its access from Pinterest settings) ends all data

collection. Stored data can be deleted at any time by the operators; to request deletion,

email the contact above.
